## Chat Web

Chat Web simple hecho con MongoDB, React.js, Node.js y Socket.io

### Setup .env
```
MONGODB_URI=
PORT=
JWT_SECRET=
NODE_ENV=production
CLOUDINARY_CLOUD_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=
```
